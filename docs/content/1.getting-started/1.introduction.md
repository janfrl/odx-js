## Core Philosophies

- **Type Safety First**: Automated model generation from your actual SAP schema.
- **Agnostic Heart**: Core logic works in any TypeScript environment, not just Nuxt.
- **Offline Ready**: Toggle between live backends and local JSON mocks seamlessly.

## Quick Start (Nuxt)

If you are building a Nuxt 4 application, you can make your first request in minutes.

### 1. Install
```bash
pnpm add @bc8-odx/nuxt
```

### 2. Configure
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@bc8-odx/nuxt'],
  odata: {
    services: [{ name: 'V2Service', url: 'https://...' }]
  }
})
```

### 3. Fetch Data
```vue [app.vue]
<script setup lang="ts">
const { data } = await useOData().V2Service.Products.list()
</script>
```
