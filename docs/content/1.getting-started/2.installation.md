# Installation

To get started with ODX in your Nuxt project, you need to install the Nuxt module package.

## Package Installation

::code-group
  ```bash [pnpm]
  pnpm add @bc8-odx/nuxt
  ```
  ```bash [npm]
  npm install @bc8-odx/nuxt
  ```
  ```bash [yarn]
  yarn add @bc8-odx/nuxt
  ```
::

## Configuration

Add `@bc8-odx/nuxt` to your `modules` section in `nuxt.config.ts`.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: [
    '@bc8-odx/nuxt'
  ],
  odata: {
    services: [
      {
        name: 'MyService',
        url: 'https://your-sap-backend.com/sap/opu/odata/sap/API_NAME',
      }
    ]
  }
})
```

::tip
ODX automatically detects the OData version (V2 or V4) and configures the server-side proxy for you.
::
