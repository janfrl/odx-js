# Introduction

**nuxt-sap-odata** provides a seamless way to integrate SAP OData services into your Nuxt 4 applications.

## Why this toolkit?

Standard OData integration often involves repetitive boilerplate and a lack of type safety. This toolkit automates the friction:

::code-group
  ```bash [pnpm]
  pnpm add nuxt-sap-odata
  ```
  ```bash [npm]
  npm install nuxt-sap-odata
  ```
  ```bash [yarn]
  yarn add nuxt-sap-odata
  ```
::

## Main Components

- **Core:** The foundation of the toolkit, containing the OData client and type definitions.
- **Proxy:** A Nitro server handler that proxies requests to your SAP backend.
- **Nuxt Module:** Integrates the core and proxy into Nuxt, providing composables and DevTools.
- **Explorer:** A custom DevTools UI for exploring your OData services.
