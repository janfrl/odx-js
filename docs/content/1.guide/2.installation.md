## Nuxt Module {#nuxt-module}

The Nuxt module is the "all-in-one" package. It bundles the proxy, the type generator, and the explorer into a single dependency.

### 1. Install
::code-group
  ```bash [pnpm]
  pnpm add @bc8-odx/nuxt
  ```
  ```bash [npm]
  npm install @bc8-odx/nuxt
  ```
::

### 2. Configure
Add the module to your `nuxt.config.ts`.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@bc8-odx/nuxt'],
  odata: {
    services: [{ name: 'MyService', url: '...' }]
  }
})
```

## Standalone Proxy {#standalone-proxy}

If you are building a custom Node.js middleware or want to keep your proxy logic separate from your frontend, use the `@bc8-odx/proxy` package.

### 1. Install
::code-group
  ```bash [pnpm]
  pnpm add @bc8-odx/proxy h3
  ```
  ```bash [npm]
  npm install @bc8-odx/proxy h3
  ```
::

### 2. Implementation
The proxy is a standard **H3** event handler. You can mount it on any server path.

```ts [server.ts]
import { createApp, createRouter, eventHandler } from 'h3'
import { createODataHandler } from '@bc8-odx/proxy'

const app = createApp()
const router = createRouter()

// Configuration for your services
const config = {
  services: [{ name: 'Northwind', url: '...' }],
  basePath: '/api/odata'
}

router.use('/api/odata/**', eventHandler(createODataHandler(config)))
app.use(router)
```
