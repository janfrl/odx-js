A high-fidelity DevTools UI for OData debugging and exploration. Built with **Nuxt UI**, it provides deep introspection into your SAP services.

## Features

- **Traffic Monitor:** Live feed of every OData request and response passing through the proxy.
- **Schema Visualizer:** Interactive graph view of your EDMX entity relationships and navigation properties.
- **Entity Browser:** Table view of your data with JSON inspection and delete capabilities.
- **Generation Trigger:** Manually trigger SDK re-generation directly from the UI.
