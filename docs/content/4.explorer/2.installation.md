The explorer is primarily used via the [Nuxt Integration](/packages/nuxt), but can be embedded as a standalone Vue component.

### 1. Install
```bash
pnpm add @bc8-odx/explorer
```

### 2. Usage
```vue
<script setup>
import { ODataExplorer } from '@bc8-odx/explorer'
</script>

<template>
  <ODataExplorer />
</template>
```
