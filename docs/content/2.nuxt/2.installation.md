The Nuxt module is the "all-in-one" solution for developers building applications with Nuxt 4.

::steps{level="3"}
### Install the package
```bash
pnpm add @bc8-odx/nuxt
```

### Register in Nuxt Config
Add the module to your `modules` array.
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@bc8-odx/nuxt']
})
```

### Configure OData Services
Define your service endpoints in the `odata` block.
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  odata: {
    services: [
      {
        name: 'MyService',
        url: 'https://...'
      }
    ]
  }
})
```
::

::caution
**ODX is pure ESM.** Ensure your project has `"type": "module"` in `package.json`.
::
