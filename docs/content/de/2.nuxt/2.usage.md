ODX bietet einen reaktiven, typsicheren Wrapper um Nuxts `useFetch` namens `useOData`.

## Nutzungsmuster

:::tabs{.w-full}
  :::tabs-item{label="Punkt-Notation" icon="i-lucide-mouse-pointer-2"}
  Der empfohlene Weg, um auf Ihre OData-Services zuzugreifen. Es bietet vollständige IDE-Autovervollständigung für Service- und Entitätsnamen.

  ```vue [app.vue]
  <script setup lang="ts">
  const { data } = await useOData().V2Service.Products.list({
    $select: ['ProductID', 'ProductName'],
    $filter: 'UnitPrice gt 20'
  })
  </script>
  ```
  :::

  :::tabs-item{label="Funktional" icon="i-lucide-code"}
  Verwenden Sie dies für dynamische Service- oder Entitätsnamen (z. B. aus Routen-Parametern).

  ```ts
  const service = 'V2Service'
  const entity = 'Products'
  const { data } = await useOData(service).entitySet(entity).list()
  ```
  :::
:::

## Verfügbare Methoden

Jedes Entitäts-Set bietet die folgenden CRUD-Methoden:

:::field-group
  :::field{name=".list(query?, options?)" type="GET"}
  Ruft eine Sammlung von Entitäten ab.
  :::

  :::field{name=".get(id, query?, options?)" type="GET"}
  Ruft eine einzelne Entität über ihren Primärschlüssel ab.
  :::

  :::field{name=".create(payload)" type="POST"}
  Erstellt eine neue Entität.
  :::

  :::field{name=".update(id, payload)" type="PATCH"}
  Aktualisiert eine bestehende Entität.
  :::

  :::field{name=".remove(id)" type="DELETE"}
  Entfernt eine Entität.
  :::
:::

## Abfrageparameter

Die Methoden `.list()` und `.get()` akzeptieren ein `ODataQuery` Objekt:

:::field-group
  :::field{name="$select" type="string | string[]"}
  Felder, die in die Antwort aufgenommen werden sollen.
  :::

  :::field{name="$filter" type="string"}
  OData-Filterausdruck (z. B. `Price gt 10`).
  :::

  :::field{name="$orderby" type="string"}
  Sortierreihenfolge (z. B. `Price desc, Name asc`).
  :::

  :::field{name="$top" type="number"}
  Begrenzt die Anzahl der zurückgegebenen Ergebnisse.
  :::

  :::field{name="$skip" type="number"}
  Anzahl der zu überspringenden Ergebnisse für die Paginierung.
  :::

  :::field{name="$expand" type="string"}
  Navigations-Properties, die in die Antwort aufgenommen werden sollen.
  :::
:::
