## Der `$odata` Client

Ein leichtgewichtiger, Promise-basierter Client, der in jeder TypeScript-Umgebung verwendet werden kann.

```ts

// Liste mit Abfrageoptionen

  .entitySet('Products')
  .list({ $select: ['ProductID'], $top: 5 })

// Abruf ĂĽber SchlĂĽssel

  .entitySet('Products')
  .get(1)

// Erstellen

  .entitySet('Products')
  .create({ ProductName: 'Neues Produkt' /* ... */ })
```
