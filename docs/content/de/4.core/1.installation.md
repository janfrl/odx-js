`@bc8-odx/core` ist ein Framework-agnostisches Paket. Verwenden Sie es, wenn Sie Standard-OData-TypeScript-Interfaces oder den leichtgewichtigen `$odata` Fetch-Client außerhalb einer Nuxt-Anwendung benötigen.

:::steps{level="2"}

## Paket installieren

:::code-group
```bash [pnpm]
pnpm add @bc8-odx/core
```
```bash [npm]
npm install @bc8-odx/core
```
```bash [yarn]
yarn add @bc8-odx/core
```
:::

## Typen importieren

Verwenden Sie die gemeinsam genutzten OData-Typen in jedem TypeScript-Projekt.

```ts
import type { ODataQuery, ODataResponse } from '@bc8-odx/core'

const query: ODataQuery = {
  $select: ['ProductID', 'ProductName'],
  $filter: 'UnitPrice gt 20',
  $top: 10
}
```

## Den `$odata` Client verwenden

Der `$odata` Client ist ein leichtgewichtiger, Promise-basierter HTTP-Client für OData-Services.

```ts
import { $odata } from '@bc8-odx/core'

const products = await $odata('https://services.odata.org/V2/Northwind/Northwind.svc/')
  .entitySet('Products')
  .list({ $top: 5 })
```

:::

## Wann Core direkt verwendet werden sollte

| Situation | Empfohlenes Paket |
| --- | --- |
| Erstellung einer Nuxt 4 Anwendung | [`@bc8-odx/nuxt`](/de/nuxt/getting-started) |
| Erstellung einer BFF-Proxy-Schicht | [`@bc8-odx/proxy`](/de/proxy/installation) |
| Schreiben von geteilten OData-Typen oder Utilities | `@bc8-odx/core` |
| Integration von OData in eine React/Plain Node App | `@bc8-odx/core` |
