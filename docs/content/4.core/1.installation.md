`@bc8-odx/core` is a framework-agnostic package. Use it when you need standard OData TypeScript interfaces or the lightweight `$odata` fetch client outside of a Nuxt application.

## 1. Install

```bash
pnpm add @bc8-odx/core
```

## 2. Import Types

Use the shared OData types in any TypeScript project.

```ts
import type { ODataQuery, ODataResponse } from '@bc8-odx/core'

const query: ODataQuery = {
  $select: ['ProductID', 'ProductName'],
  $filter: 'UnitPrice gt 20',
  $top: 10
}
```

## 3. Use the `$odata` Client

The `$odata` client is a lightweight, promise-based HTTP client for OData services.

```ts
import { $odata } from '@bc8-odx/core'

const products = await $odata('https://services.odata.org/V2/Northwind/Northwind.svc/')
  .entitySet('Products')
  .list({ $top: 5 })

console.log(products)
```

## When to Use Core Directly

| Situation | Recommended Package |
| --- | --- |
| Building a Nuxt 4 application | [`@bc8-odx/nuxt`](/nuxt/getting-started) |
| Building a BFF proxy layer | [`@bc8-odx/proxy`](/proxy/installation) |
| Writing shared OData types or utilities | `@bc8-odx/core` ← you are here |
| Integrating OData in a React/SvelteKit/plain Node app | `@bc8-odx/core` |
