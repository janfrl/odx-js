`@bc8-odx/core` is a framework-agnostic package containing the core OData logic, types, and the base `$odata` client.

## Key Interfaces

### `ODataQuery`

Used to define structured queries for OData services.

| Property | Type | Description |
| --- | --- | --- |
| `$select` | `string \| string[]` | Fields to include in the response. |
| `$filter` | `string` | OData filter expression. |
| `$orderby` | `string` | Sort order. |
| `$top` | `number` | Limit result count. |
| `$skip` | `number` | Offset for pagination. |
| `$expand` | `string` | Navigation properties to include. |

## The `$odata` Client

A lightweight, promise-based client that can be used in any TypeScript environment.

```ts
import { $odata } from '@bc8-odx/core'

const products = await $odata('https://.../V2Service').entitySet('Products').list()
```

## Internal Utilities

The package also exports various utilities for:
- CSRF token management
- URL building and query serialization
- SAP-specific header handling
