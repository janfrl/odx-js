The ODX Explorer is a specialized UI integrated into Nuxt DevTools, designed to help you introspect and debug your OData services.

## Features

### Schema Visualizer
Visualize the entity relationships, properties, and navigation paths of your OData services. Supports both V2 and V4 schemas.

### Traffic Monitor
Monitor live OData requests triggered by your application. See the raw URL, headers, and payload, as well as the response from the backend.

### Data Browser
Browse and filter entity data directly within the DevTools. Perfect for verifying that your filters and queries are working as expected.

## Integration

The Explorer is automatically enabled when the `@bc8-odx/nuxt` module is used in development mode.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  odata: {
    devtools: {
      enabled: true,
      maxLogs: 100
    }
  }
})
```

For more details on how to use the Explorer, see the [Explorer Setup Guide](/how-to/explorer-setup).
