## Composables

### `useOData`

The main composable for fetching data.

```ts
const { data, pending, error, status } = await useOData().ServiceName.EntitySet.list(query, options)
```

- **Returns**: `ODataAsyncData<T>` (reactive object similar to Nuxt's `AsyncData`)
- **Params**:
  - `query`: `ODataQuery` object for filtering, sorting, etc.
  - `options`: Standard Nuxt `useFetch` options.
