Use the standalone proxy for a Backend-for-Frontend (BFF) pattern, a microservice architecture, or any framework other than Nuxt.

:::steps{level="3"}
### Install dependencies

```bash
pnpm add @bc8-odx/proxy h3
```

### Create the handler

Use the `createODataHandler` utility to build a standard H3 event handler.

```ts [server.ts]
import { createODataHandler } from '@bc8-odx/proxy'

const config = {
  services: [
    { name: 'Northwind', url: 'https://services.odata.org/V2/Northwind/Northwind.svc/' }
  ]
}

export const odataHandler = createODataHandler(config)
```

### Mount the route

The handler is compatible with any standard H3 or Nitro server.

```ts [nitro.config.ts]
export default defineNitroConfig({
  modules: ['@bc8-odx/proxy/nitro'],
  odata: {
    services: [
      { name: 'MyService', url: '...', destination: 'MY_DESTINATION' }
    ]
  }
})
```
:::

## Configuration Options

The proxy configuration object mirrors the `odata` block of the Nuxt module. All [service definition options](/nuxt/module-reference#odataservices) from the Nuxt module apply here.
