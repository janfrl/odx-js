Use the standalone proxy for Backend-for-Frontend (BFF) or middleware services.

::steps{level="3"}
### Install dependencies
```bash
pnpm add @bc8-odx/proxy h3
```

### Create the handler
Use the `createODataHandler` utility.
```ts [server.ts]
import { createODataHandler } from '@bc8-odx/proxy'

const config = {
  services: [{ name: 'Northwind', url: '...' }]
}

const handler = createODataHandler(config)
```

### Mount the route
The proxy works with any standard H3 or Express server.
::
