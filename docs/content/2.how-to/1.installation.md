Integrating ODX into your Nuxt 4 project is straightforward. Follow these steps to get started.

## 1. Install Dependencies

Add the `@bc8-odx/nuxt` package to your project using your preferred package manager.

```bash
pnpm add @bc8-odx/nuxt

npm install @bc8-odx/nuxt

yarn add @bc8-odx/nuxt
```

## 2. Register the Module

Add the module to the `modules` array in your `nuxt.config.ts`.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: [
    '@bc8-odx/nuxt'
  ]
})
```

## 3. Prepare for Development

Run the `prepare` command to generate the necessary type stubs and internal ODX directories.

```bash
npx nuxi prepare
```

This will create a `.odx` directory in your project root, which will store the generated TypeScript models and EDMX metadata.

## 4. Next Steps

Now that ODX is installed, you need to [Configure your OData services](/how-to/configuration) to start fetching data.
